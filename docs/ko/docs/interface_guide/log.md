# 로그

웹 관리 패널 왼쪽에서 **SYSTEM** -> **Log**로 이동합니다.

로그 페이지에서 분석 및 문제 해결을 위해 시스템, 커널, 충돌, 클라우드 및 Nginx 로그를 확인할 수 있습니다.

eSIM을 지원하는 일부 셀룰러 모델은 eSIM 로그를 제공합니다.

GL-X3000의 예입니다.

![log page](https://static.gl-inet.com/docs/router/en/4/interface_guide/log/log.png){class="glboxshadow"}

**필터링** 기능은 펌웨어 v4.8부터 시스템 로그 페이지에서 사용할 수 있습니다. 수준 또는 모듈을 선택하거나 키워드를 입력하여 로그를 빠르게 쿼리할 수 있습니다.

**새로고침**을 클릭하여 최신 로그 정보를 가져옵니다.

**로그 내보내기**를 클릭하여 시스템, 커널, 충돌 및 클라우드의 로그 정보를 내보냅니다. GL.iNet에 피드백을 제공할 때 내보낸 로그 파일을 GL.iNet 기술 지원에 보내면 더 빠른 문제 분석이 가능합니다.
