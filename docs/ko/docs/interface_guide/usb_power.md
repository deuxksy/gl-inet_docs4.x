# USB 및 전원

**참고**: 이 페이지는 **GL-E5800 (Mudi 7)**에서만 사용할 수 있습니다.

웹 관리 패널 왼쪽�에서 **SYSTEM** -> **USB & Power**로 이동합니다.

이 페이지에서 USB 프로토콜, 전원 방향, 유휨 시간 초과, 대기 모드 동작과 같은 USB 관련 설정과 전원 관리를 구성할 수 있습니다.

## USB 설정

이 섹션에서 라우터 USB 포트의 동작을 사용자 정의할 수 있습니다.

![usb settings](https://static.gl.inet.com/docs/router/en/4/interface_guide/usb_power/e5800_usb.png){class="glboxshadow"}

- **USB 프로토콜 스위치**: USB 포트에 대해 USB 2.0과 USB 3.1 프로토콜 사이를 전환합니다.

- **듀얼 역할 USB 모드**: 드롭다운 메뉴에서 USB 작동 모드를 선택합니다. Device 또는 Host로 설정할 수 있습니다.

    ![usb role](https://static.gl.inet.com/docs/router/en/4/interface_guide/usb_power/usb_role.png){class="glboxshadow"}

- **전원 방향**: 드롭다운 메뉴에서 USB 포트에 대한 전원 우선 순위를 선택합니다. Input Priority 또는 Output Priority로 설정할 수 있습니다.

- **전원 임계값**: USB 작업에 필요한 최소 전원 수준을 제어하기 위해 USB 포트에 대한 특정 전원 임계값 백분율을 설정하여 배터리 수명을 연장합니다.

## 전원 설정

이 섹션에서 전력 소비와 장치 동작을 최적화할 수 있습니다.

![power settings](https://static.gl.inet.com/docs/router/en/4/interface_guide/usb_power/e5800_power.png){class="glboxshadow"}

- **Wi-Fi 유휨 시간 제한**: Wi-Fi가 대기 상태로 들어갈 때까지의 유휨 지속 시간을 설정합니다(10분~2시간 또는 사용 안 함).

- **이더넷 유휨 시간 제한**: 이더넷이 대기 상태로 전환될 때까지의 유휨 지속 시간을 설정합니다(10분~2시간 또는 사용 안 함).

- **전원 꺼기 시간 제한**: 유휨 상태일 때 라우터가 자동으로 전원이 꺼지는 시간 지연을 설정합니다(1시간~12시간 또는 사용 안 함).

- **충전기로 전원 켜기**: 충전기에 연결될 때 라우터가 자동으로 전원이 켜지도록 활성화/비활성화합니다.

- **자동 대기**: 전원 절약을 위해 자동 대기를 활성화/비활성화합니다.
