# UI 언어

이 기능은 펌웨어 v4.6부터 추가되었습니다.

---

우측 상단의 언어 버튼을 클릭하여 UI 언어를 전환하거나 추가할 수 있습니다.

추가 UI 언어를 추가하거나 UI 텍스트를 업데이트하려면 **Add Languages**를 클릭하세요.

![languages list](https://static.gl.inet.com/docs/router/en/4/interface_guide/ui_languages/add_languages.jpg){class="glboxshadow"}

팝업 창�에서 원하는 언어를 선택하고 **Update**를 클릭하세요. 라우터가 자동으로 저장소에서 최신 언어 팩을 가져옵니다. 설치되지 않은 언어 팩은 제거됩니다(영어 제외).

텍스트 문제를 수정하기 위해 최신 언어 팩을 자동으로 가져오려면 **Enable Auto Update**를 토글하고 업데이트 시간을 설정하세요. 그러면 라우터가 지정된 시간에 자동으로 최신 언어 팩을 다운로드합니다.

![languages list](https://static.gl.inet.com/docs/router/en/4/interface_guide/ui_languages/languages_list.png){class="glboxshadow"}
