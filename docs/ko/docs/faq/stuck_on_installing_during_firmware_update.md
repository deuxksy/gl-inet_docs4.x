# 펌웨어 업데이트 중 "설치 중"에서 멈춤

펌웨어 업데이트 중 라우터가 "설치 중"에서 멈추가 된 경우 다음 단계에 따라 문제를 해결하세요.

![stuck on installing](https://static.gl.inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_installing/installing.png){class="glboxshadow"}

각 단계 후 문제가 해결되었는지 확인하세요. 해결되면 나머지 단계를 건너뛰세요.

1. 펌웨어 업데이트가 성공했는지 확인하세요.

    먼저 웹 브라우저를 열고 `192.168.8.1`에 액세스한 후 로그인하세요.

    그런 다음 왼쪽 상단의 펌웨어 버전을 확인하세요. 대상 버전과 일치하면 업데이트가 성공한 것입니다. 그렇지 않으면 다음 단계로 진행하세요.

2. 안정적인 네트워크 연결을 확인한 후 펌웨어를 다시 업로드하세요.

3. 다른 브라우저(예: Edge, Safari)를 사용하여 펌웨어를 다시 업로드하세요.
