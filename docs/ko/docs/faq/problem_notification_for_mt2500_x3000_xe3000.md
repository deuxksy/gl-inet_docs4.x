# GL-MT2500/GL-X3000/GL-XE3000에 대한 심각한 문제 알림

친애하는 GL.iNet 고객 여러분,

이 공지는 GL-MT2500/GL-X3000/GL-XE3000 장치에서 U-Boot를 통한 시작 실패 및 펌웨어 복구 문제를 다룹니다. 이 문제는 일부 매개변수가 잘못 정의된 소프트웨어 결함에서 비롯되었으며, 이로 인해 라우터가 비정상적으로 작동했습니다. 결과적으로 우리는 이 문제가 라우터의 서비스 수명을 어느 정도 단축시킬 수 있다고 평가했으며, 이에 대해 진심으로 사과드립니다.

이 문제를 완전히 해결하기 위해 GL-MT2500, GL-X3000 및 GL-XE3000의 모든 사용자가 U-Boot 부트로더 업그레이드와 펌웨어 업그레이드를 모두 수행할 것을 강력히 권장합니다. 두 업그레이드 절차都是 필수적이며 생략해서는 안 됩니다.

장치가 이미 최신 펌웨어로 업그레이드되었더라도 잠재적인 문제를 방지하기 위해 U-Boot 부트로더 업그레이드는 여전히 필수입니다. U-Boot 부트로더 업그레이드를 완료한 후 최신 펌웨어가 이미 설치되어 있으면 펌웨어를 다시 설치하지 않아도 됩니다.

Chrome 또는 Microsoft Edge를 사용하여 업그레이드하세요. Mozilla/Firefox는 사용하지 마세요.

## GL-MT2500

1. U-Boot 부트로더를 [버전 2025-02-24 15:04:07](https://github.com/gl-inet/mt798x-boot/blob/main/bin/uboot-mt2500-20250224-md5-74286e770cfb041b611d80d4adaef189.bin){target="_blank"}(으)로 다운로드하고 업그레이드하세요.

2. 펌웨어를 [버전 4.7.4](https://fw.gl-inet.com/firmware/mt2500/release/openwrt-mt2500-4.7.4-0328-1743128340.bin)로 다운로드하고 업그레이드하세요.

## GL-X3000

1. U-Boot 부트로더를 [버전 2025-02-25 19:05:24](https://github.com/gl-inet/mt798x-boot/blob/main/bin/uboot-x3000-20250225-md5-c9d7b2fd2451adbc0bb126e2d9729e87.bin){target="_blank"}(으)로 다운로드하고 업그레이드하세요.

2. 펌웨어를 [버전 4.7.4](https://fw.gl-inet.com/firmware/x3000/release/openwrt-x3000-4.7.4-0317-1742206344.bin)로 다운로드하고 업그레이드하세요.

## GL-XE3000

1. U-Boot 부트로더를 [버전 2025-02-25 19:07:14](https://github.com/gl-inet/mt798x-boot/blob/main/bin/uboot-xe3000-20250225-md5-05fadd9da27314d41dbadc6fbd239b3d.bin){target="_blank"}(으)로 다운로드하고 업그레이드하세요.

!!! Note

    U-Boot 부트로더 업그레이드 방법은 [이 링크](https://docs.gl-inet.com/router/en/4/faq/upgrade_uboot_version/){target="_blank"}를 참조하세요.

    펌웨어 업그레이드 방법은 [이 링크](https://docs.gl-inet.com/router/en/4/tutorials/how_to_upgrade_downgrade_router/){target="_blank"}를 참조하세요.
