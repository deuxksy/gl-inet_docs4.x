# GL.iNet 라우터 터미널에 원격 SSH 접속 방법은 무엇인가요?

- **방법 1**: 라우터의 네트워크에 [공개 IP](../tutorials/how_to_check_if_isp_assigns_you_a_public_ip_address.md)가 있는 경우 [WireGuard 서버](../interface_guide/wireguard_server.md)를 설정하고 WireGuard를 사용하여 라우터에 SSH 접속하세요. 공개 IP가 없는 경우 아래 다른 방법을 시도하세요.
- **방법 2**: [GoodCloud에 장치를 바인딩](../interface_guide/cloud.md#setup)한 다음 [GoodCloud 웹사이트](https://www.goodcloud.xyz){target="_blank"}에 로그인하여 [라우터 터미널에 원격으로 SSH 접속](../interface_guide/cloud.md#remote-access-routers-terminal)할 수 있습니다.
- **방법 3**: SDWAN 솔루션을 사용하려면 [AstroWarp](https://www.astrowarp.net){target="_blank"}을 권장합니다.
- **방법 4**: 리버스 프록시 솔루션을 사용하려면 [AstroRelay](https://www.astrorelay.com){target="_blank"}을 권장합니다.
